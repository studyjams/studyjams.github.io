常见问题

  • 什么是 Google 开发者学习班？

    Google 开发者学习班为期8周，由社区组织，包括一系列的现场学习活动，帮助您通过 Udacity 课程学习 Google 开发者技术。这些活动分布全球，由当地的 Google (或其他)开发者社区组织。本季学习班将围绕 Udacity 课程，Developing Android Apps: Android Fundamentals 进行。

  • 学习班上会发生什么？

    学习班是一个有趣的课堂，您可以免费获得 Udacity 课程的个人指导。学习班围绕 Udacity 的课程结构而设计。每周一节课，每节课后（共6课，第4课分ABC三讲），一次线下活动。另加两个星期完成和分享大作业。学员应来提前完成每周的 Udacity 线上课程，这样才能利用线下活动复习重要概念，解决问题，或开发单独的项目。现场的社区氛围和辅导旨在帮助学生深入学习课程中讲述的概念。

  • 加入学习班有什么要求？

    欢迎各界人士参加学习班。学员应该首先符合课程的先决条件和要求： 开发 Android 应用：Android 基础课是为具有至少3年 Java 或其他面向对象编程经验的学员开设的。 需要下载 Android Studio，在整个过程中都需要使用。您还应该能熟练使用 Github 管理代码 。有一部 Android 设备会有帮助，但不是完成大作业所必需的。请参阅 Technology Requirements 了解如何使用 Udacity.

  • 学习班什么时候开课？

    围绕 Android 基础课的学习班学期开始时间，2015年2月1日，结束时间4月31日。（中国和其他过春节的国家和地区可延长一个月。）

  • 对 Udacity 的 Android 基础课，我想了解更多

    请查看 Android 基础课页面 。

  • 如何报名？
    1. 在地图上查看活动，找到您附近的学习班。点击地图上的圆点，报名当地的活动。
    2. 报名 Udacity 的 Android 基础课：登录 Udacity, 加入学习班社区，查找“开发 Android 应用：Android 基础课”（可以搜索“Android Fun”）。然后点击蓝色按钮“获取教材”。这里有完整的报名步骤 。
  • 如何报名 Udacity 的课程？

    请直接使用这个专用的学习班链接来报名 Android 基础课，参考这些步骤 。

  • 我可以在 Udacity 网站上直接报名吗？

    我们要求您使用我们提供的 Udacity 报名链接。这样，我们才能跟踪您的课程学习进度，并为您颁发课程结业徽章。

  • 我已经有 Udacity 账户，或以前报过 Android 基础课，该怎么办？

    好极了！还是要请您使用学习班报名链接 ，这样我们才能知道你参加了本系列课程。

  • 我要为 Udacity 课程付费吗？

    不需要。学习班使用本课的免费教程。课程报名后，点击蓝色的按钮，可以获取教材。请参考报名过程的指导。

  • 完成课程，并参加学习班所有线下活动之后，能否获得课程证书？

    参加学习班所有线下活动，并完成大作业的学员将获得学习班结业徽章。这个徽章是从官方的Android Udacity基础课程认证不同。此徽章不同于正式的 Udacity Android 基础课证书。请参考下面的“如何获得课程证书？”以了解更多信息。

  • 如何获得课程证书？

    所有参加学习班的学员，如果完成了 Udacity 课程和大作业，向 Udacity 提交大作业以获取正式课程证书时，可以享受免除20%费用的待遇。收费标准通常是每月$199。学员有机会免费提交大作业，如果他们：1）参加学习班所有的线下活动，2）完成Udacity 课程，3）完成大作业。

  • 对时间有什么要求？

    学习班线下活动每周一次，持续8到9周，每次1-2个小时。学习班是对 Udacity 在线课程的补充。该课程平均每周需要花费6小时，但因人而已。

  • 不能参加所有的线下活动怎么办？

    我们鼓励学员参加所有的线下活动，来获得对学习般的最充分体验。出席所有学习班线下活动，是获得结业徽章的必要条件。但是，如果您无法出席一两次活动，也别太担心。

  • 你们为什么要做这件事？

    我们认为，利用活跃的社区学习氛围来补充在线课程，可以帮助学员完成课程并获取证书。我们的目标是在一个有趣的和协作的社区环境中，更深入的理解教材。

  • Udacity 课程是如何运作的？

    请查看 Udacity FAQs .

  • 1280 x 800 minimum screen resolution
  • Java Runtime Environment (JRE) 6

Tested on Ubuntu® 14.04, Trusty Tahr (64-bit distribution capable of running 32-bit applications).

Linux 用户可以使用下面的命令行下载和解压推荐的 JDK 版本, 并设置成操作系统默认的 JDK

wget http://studyjams.dushu.hu/jdk7/jdk-7u75-linux-x64.gz -O - | sudo tar -zxf - -C /opt
sudo update-alternatives --install "/usr/bin/java" java "/opt/jdk1.7.0_75/bin/java" 1
sudo update-alternatives --config java

Linux 用户如果看到 HAXM is required for running x86 System Images, 可以忽略。

建议不要使用32位 Linux 操作系统
